Use case name
CheckSickness
Participating Actors
Started by User
Flow of events
1. User selects "Request sickness allowance."
2. The system displays the page with the information for requesting the INPS office for the sickness check, also allowing the user to print it out.
Entry condition
The user has accessed the system and selected the Social Assistance feature.
Exit condition
The user viewed the page, OR The system encountered an unanticipated error then called the use case
ServiceNotAvailable.
